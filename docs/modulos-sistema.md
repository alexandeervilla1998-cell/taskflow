# Módulos del sistema TaskFlow

## Objetivo

TaskFlow es una aplicación web para gestionar tareas personales mediante una arquitectura cliente servidor.

## 1. Autenticación

### Funcionalidades

* Inicio de sesión
* Registro de usuarios
* Cierre de sesión
* Protección de rutas mediante JWT

## 2. Dashboard

### Funcionalidades

* Resumen de tareas
* Estadísticas generales
* Accesos rápidos
* Visualización de tareas recientes

## 3. Gestión de tareas

### Funcionalidades

* Crear tarea
* Editar tarea
* Eliminar tarea
* Consultar tarea
* Buscar tarea
* Filtrar tarea
* Ordenar tarea
* Cambiar estado

## 4. Gestión de categorías

### Funcionalidades

* Crear categoría
* Editar categoría
* Eliminar categoría
* Asignar categoría a una tarea

## 5. Perfil de usuario

### Funcionalidades

* Ver perfil
* Editar información personal
* Actualizar correo electrónico
* Cambiar contraseña
* Cambiar foto de perfil

## 6. Administración

### Funcionalidades

* Gestión de usuarios
* Gestión de roles
* Configuración general del sistema

## Tecnologías

### Frontend

* React
* React Router
* Axios
* Bootstrap

### Backend

* Spring Boot
* Spring Security
* JWT
* Spring Data JPA

### Base de datos

* MySQL

### Control de versiones

* Git
* GitHub

### Despliegue

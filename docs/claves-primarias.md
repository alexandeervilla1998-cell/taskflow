# Claves Primarias de TaskFlow

## Tabla usuarios

Clave primaria:

* id

Características:

* BIGINT
* AUTO_INCREMENT
* NOT NULL
* UNIQUE

## Tabla categorias

Clave primaria:

* id

Características:

* BIGINT
* AUTO_INCREMENT
* NOT NULL
* UNIQUE

## Tabla tareas

Clave primaria:

* id

Características:

* BIGINT
* AUTO_INCREMENT
* NOT NULL
* UNIQUE

## Resumen

Todas las tablas utilizan una clave primaria simple llamada "id".

El identificador es numérico, único y generado automáticamente por la base de datos.
